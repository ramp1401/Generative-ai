🚀 Built My First GenAI App using LangChain + Ollama + Streamlit!

Excited to share a simple yet powerful project I’ve been working on 👇

🔹 Created an AI-powered Q&A app using:

* LangChain for chaining prompts and LLMs
* Ollama (gemma:2b) for running LLM locally
* Streamlit for building an interactive UI

💡 Key Features:
✔️ Clean prompt engineering using ChatPromptTemplate
✔️ Local LLM execution (no API cost!)
✔️ Simple and interactive user interface
✔️ End-to-end pipeline: Prompt → LLM → Output Parser

🧠 What I Learned:

* How LangChain chains work (`prompt | llm | parser`)
* Using `invoke()` for real-time responses
* Structuring GenAI apps for production thinking
* Integrating LLMs into web apps

📌 This is just the beginning — next I’m planning to build:
➡️ RAG-based apps
➡️ Memory-enabled chatbots
➡️ Multi-agent systems

If you're learning Generative AI, I highly recommend building projects like this!
